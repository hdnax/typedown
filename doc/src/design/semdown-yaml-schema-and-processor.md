# Semdown YAML Schema & Processor

Semdown YAML Schema is just YAML.
- A valid Semdown YAML is a valid YAML.
- A YAML using the features as defined by the spec YAML 1.2.2 & the built-in "core" schema are processable by the Semdown YAML processor.

Semdown YAML processor extends the YAML built-in core schemas with many more tags for better content management.
