# Glossary

## Graph Database

| Term                   | Definition                                                                                                | See                                                                                                                                                                                   |
| ---------------------- | --------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Cypher                 | Query language for property graph databases                                                               | [The Graph Space](./graph-database/graph-databases-book/3-the-graph-space.md)                                                                                                         |
| Global Graph Algorithm | An algorithm that operates on the entire graph (e.g. PageRank, community detection, connected components) | [Local vs Global](./graph-database/graph-databases-book/5-concept-local-vs-global-graph-operations.md)                                                                                |
| Aggregate NoSQL Store  | NoSQL database organizing data around self-contained chunks (key-value, document, or column-family)       | [Modeling Connectivity](./graph-database/graph-databases-book/8-modeling-connectivity.md)                                                                                             |
| Graph                  | A collection of nodes and relationships connecting them                                                   | [The Concept of Graph](./graph-database/graph-databases-book/1-concept-graph.md)                                                                                                      |
| Graph Compute Engine   | An engine for offline batch analytics over graph data (like OLAP)                                         | [The Graph Space](./graph-database/graph-databases-book/3-the-graph-space.md), [Graph Compute Engine](./graph-database/graph-databases-book/6-concept-graph-compute-engine.md)        |
| Graph Database         | An online CRUD database exposing a graph data model, optimized for OLTP                                   | [The Graph Space](./graph-database/graph-databases-book/3-the-graph-space.md), [Graph Database](./graph-database/graph-databases-book/4-concept-graph-database.md)                    |
| Hypergraph             | A graph model where a single edge can connect more than two nodes                                         | [The Graph Space](./graph-database/graph-databases-book/3-the-graph-space.md)                                                                                                         |
| Index-free Adjacency   | Connected nodes physically point to each other, O(1) traversal per hop                                    | [Graph Database](./graph-database/graph-databases-book/4-concept-graph-database.md)                                                                                                   |
| Local Graph Operation  | Starts at a specific node and traverses outward, touching a small portion of the graph                    | [Local vs Global](./graph-database/graph-databases-book/5-concept-local-vs-global-graph-operations.md)                                                                                |
| Native Graph Storage   | Storage engine purpose-built for graphs (vs serializing into relational/other backends)                   | [Graph Database](./graph-database/graph-databases-book/4-concept-graph-database.md)                                                                                                   |
| Node                   | An entity in a graph, can hold properties (key-value pairs)                                               | [The Concept of Graph](./graph-database/graph-databases-book/1-concept-graph.md), [The Property Graph Model](./graph-database/graph-databases-book/2-concept-property-graph-model.md) |
| Pregel                 | Google's paper describing a distributed graph compute engine, basis for most distributed implementations  | [Graph Compute Engine](./graph-database/graph-databases-book/6-concept-graph-compute-engine.md)                                                                                       |
| Property               | A key-value pair attached to a node or relationship                                                       | [The Property Graph Model](./graph-database/graph-databases-book/2-concept-property-graph-model.md)                                                                                   |
| Property Graph         | The most popular graph model variant, nodes and relationships with properties                             | [The Property Graph Model](./graph-database/graph-databases-book/2-concept-property-graph-model.md), [The Graph Space](./graph-database/graph-databases-book/3-the-graph-space.md)    |
| Relationship (Edge)    | A named, directed connection between two nodes, can hold properties                                       | [The Concept of Graph](./graph-database/graph-databases-book/1-concept-graph.md), [The Property Graph Model](./graph-database/graph-databases-book/2-concept-property-graph-model.md) |
| Relationship Chain     | Doubly linked lists of relationships connecting a node to its neighbors on disk                           | [Native Graph Storage](./graph-database/graph-databases-book/12-native-graph-storage.md)                                                                                              |
| Write-Ahead Log        | Durability mechanism: Changes are logged before being applied to store files                              | [Transactions, Availability, Scale](./graph-database/graph-databases-book/13-internals-transactions-availability-scale.md)                                                            |

## Semantic Web

| Term                     | Definition                                                                                                                  | See                                                                                                       |
| ------------------------ | --------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------- |
| Digital Signature        | Encrypted data blocks proving information came from a trusted source                                                        | [The Semantic Web](./web-3/the-semantic-web.md)                                                           |
| Equivalence Statement    | Explicit links between different names for the same concept across databases (e.g. "zipcode" = "postal code")               | [Semantic Web](./web-3.md)                                                                                |
| Inference Rules          | Logical deductions encoded in ontologies (e.g. "if X is in city Y, and Y is in state Z, then X is in state Z")              | [Semantic Web](./web-3.md)                                                                                |
| Knowledge Representation | Representing a knowledge base in a machine-readable fashion                                                                 | [The Semantic Web](./web-3/the-semantic-web.md)                                                           |
| Ontology                 | A document formally defining relations among terms: Taxonomies, inference rules, and equivalence relations                  | [Semantic Web](./web-3.md), [The Graph Space](./graph-database/graph-databases-book/3-the-graph-space.md) |
| Proof Verification       | Verifying that claims are logically inferred from trusted sources via exchangeable mathematical proofs                      | [The Semantic Web](./web-3/the-semantic-web.md)                                                           |
| RDF                      | Resource Description Framework: Triple-based model (subject-predicate-object) encoding meaning via URIs                     | [Semantic Web](./web-3.md), [The Graph Space](./graph-database/graph-databases-book/3-the-graph-space.md) |
| Semantic Preservation    | Extracting universal logic from systems while discarding system-specific instructions, losing as little meaning as possible | [The Semantic Web](./web-3/the-semantic-web.md)                                                           |
| Semantic Web             | Extension of the WWW adding machine-readable meaning to data, enabling computer-human cooperation                           | [Semantic Web](./web-3.md)                                                                                |
| Software Agent           | Software that harvests data from diverse sources, processes it, and exchanges results with other agents                     | [The Semantic Web](./web-3/the-semantic-web.md)                                                           |
| SPARQL                   | Query language for RDF data                                                                                                 | [The Graph Space](./graph-database/graph-databases-book/3-the-graph-space.md)                             |
| Taxonomy                 | Class hierarchies (classes and subclasses) and their relations within an ontology                                           | [Semantic Web](./web-3.md)                                                                                |
| URI                      | Unique identifier tying a concept to a discoverable definition, solving ambiguity across systems                            | [Semantic Web](./web-3.md)                                                                                |
| XML                      | Adds arbitrary structure to documents via custom tags, structure without meaning                                            | [Semantic Web](./web-3.md)                                                                                |
