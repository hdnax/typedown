# URI: Global Identity

## Resources

### Spec

- [RFC 3986: URI Generic Syntax](https://www.rfc-editor.org/rfc/rfc3986) - The definitive spec. Defines the syntax of URIs (scheme, authority, path, query, fragment).
- [RFC 3987: IRI (Internationalized Resource Identifiers)](https://www.rfc-editor.org/rfc/rfc3987) - Extension of URI to support Unicode characters.

### In Semantic Web

- [W3C RDF 1.2 Concepts: Section on IRIs](https://www.w3.org/TR/rdf12-concepts/) - How URIs are used as the identity layer in RDF triples.

### Others

- [W3C: URIs, URLs, and URNs](https://www.w3.org/TR/uri-clarification/) - Clarifies the confusing URI vs URL vs URN distinction.
- [Tim Berners-Lee: Cool URIs Don't Change](https://www.w3.org/Provider/Style/URI) - Design principles for long-lived URIs. Short and opinionated.
- [Tim Berners-Lee: Linked Data Principles](https://www.w3.org/DesignIssues/LinkedData.html) - The four rules for publishing data on the web, all built on URIs.
