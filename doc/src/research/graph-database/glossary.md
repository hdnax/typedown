# Glossary

| Term                   | Definition                                                                                                       | See                                                                                                                                                     |
| ---------------------- | ---------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Graph                  | A collection of nodes and relationships connecting them                                                          | [The Concept of Graph](./graph-databases-book/1-concept-graph.md)                                                                                       |
| Node                   | An entity in a graph, can hold properties (key-value pairs)                                                      | [The Concept of Graph](./graph-databases-book/1-concept-graph.md), [The Property Graph Model](./graph-databases-book/2-concept-property-graph-model.md) |
| Relationship (Edge)    | A named, directed connection between two nodes, can hold properties                                              | [The Concept of Graph](./graph-databases-book/1-concept-graph.md), [The Property Graph Model](./graph-databases-book/2-concept-property-graph-model.md) |
| Property               | A key-value pair attached to a node or relationship                                                              | [The Property Graph Model](./graph-databases-book/2-concept-property-graph-model.md)                                                                    |
| Property Graph         | The most popular graph model variant - nodes and relationships with properties                                   | [The Property Graph Model](./graph-databases-book/2-concept-property-graph-model.md), [The Graph Space](./graph-databases-book/3-the-graph-space.md)    |
| RDF                    | Resource Description Framework - triple-based model (subject-predicate-object) for semantic web interoperability | [The Graph Space](./graph-databases-book/3-the-graph-space.md)                                                                                          |
| Hypergraph             | A graph model where a single edge can connect more than two nodes                                                | [The Graph Space](./graph-databases-book/3-the-graph-space.md)                                                                                          |
| Ontology (OWL/RDFS)    | Formal rules enabling automatic inference in RDF stores                                                          | [The Graph Space](./graph-databases-book/3-the-graph-space.md)                                                                                          |
| SPARQL                 | Query language for RDF data                                                                                      | [The Graph Space](./graph-databases-book/3-the-graph-space.md)                                                                                          |
| Cypher                 | Query language for property graph databases                                                                      | [The Graph Space](./graph-databases-book/3-the-graph-space.md)                                                                                          |
| Graph Database         | An online CRUD database exposing a graph data model, optimized for OLTP                                          | [The Graph Space](./graph-databases-book/3-the-graph-space.md), [Graph Database](./graph-databases-book/4-concept-graph-database.md)                    |
| Native Graph Storage   | Storage engine purpose-built for graphs (vs serializing into relational/other backends)                          | [Graph Database](./graph-databases-book/4-concept-graph-database.md)                                                                                    |
| Index-free Adjacency   | Connected nodes physically point to each other - O(1) traversal per hop                                          | [Graph Database](./graph-databases-book/4-concept-graph-database.md)                                                                                    |
| Graph Compute Engine   | An engine for offline batch analytics over graph data (like OLAP)                                                | [The Graph Space](./graph-databases-book/3-the-graph-space.md), [Graph Compute Engine](./graph-databases-book/6-concept-graph-compute-engine.md)        |
| Local Graph Operation  | Starts at a specific node and traverses outward, touching a small portion of the graph                           | [Local vs Global](./graph-databases-book/5-concept-local-vs-global-graph-operations.md)                                                                 |
| Global Graph Algorithm | An algorithm that operates on the entire graph (e.g. PageRank, community detection, connected components)        | [Local vs Global](./graph-databases-book/5-concept-local-vs-global-graph-operations.md)                                                                 |
| Pregel                 | Google's paper describing a distributed graph compute engine, basis for most distributed implementations         | [Graph Compute Engine](./graph-databases-book/6-concept-graph-compute-engine.md)                                                                        |
